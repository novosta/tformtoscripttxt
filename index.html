T-Form Script Converter

Convert Your T-Form Excel File to a Script

Upload an Excel file (.xlsx) and generate a clean, formatted script based on its content. The script will automatically remove unnecessary entries like "Middle Camera" and clean up any `(nan)` placeholders.

How to Use:
  • 1. Prepare your T-Form file in Excel format (.xlsx).
  • 2. Upload the file using the "Choose File" button.
  • 3. Click "Convert to Script" to generate the output.
  • 4. The script will remove unnecessary directions like "Middle Camera" and clean up `(nan)` entries.
  • 5. Download the converted script when prompted.
Exporting from Google Sheets:
  • 1. Open your Google Sheet at Google Sheets.
  • 2. Go to the File menu, select Download, then choose Microsoft Excel (.xlsx).
  • 3. Save the file to your computer, then upload it here.
Supported Features:
  • Excel (.xlsx) to script conversion.
  • Removes unnecessary fields and placeholders like `(nan)`.
  • Handles missing camera directions or dialogue fields gracefully.
